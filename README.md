# ShareSphere 🌍

ShareSphere is a community-driven donation platform designed to connect people with surplus goods to those in need (NGOs and individuals).

## Features
- **Map Discovery:** Find nearby donations and facilities.
- **NGO Connect:** Locate and contact local NGOs.
- **Donor Mode:** Easily list items for donation.

## Tech Stack
- **Frontend:** React, Tailwind CSS, Leaflet.
- **Backend:** Node.js, Express, MongoDB.

## Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB (running locally or a cloud instance)

### Installation

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   ```

2. **Backend Setup**
   ```bash
   cd backend
   npm install
   # Create a .env file and add your MONGODB_URI
   npm run dev
   ```

3. **Frontend Setup**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
